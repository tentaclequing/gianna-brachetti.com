Today, a friend - let's call them "Konny", like my pet octopus - was asked to implement "AEO code" into the HTML head. A note about the pain a lot of inhouse SEOs face these days:

What this was: 
Structured data. 

Why this bothered Konny:
They were asked to create a roadmap. Specifically: to add things for "AI SEO", "LLM SEO", "GEO" or whatever fancy acronym you want to call it these days. 

Their roadmap contained structured data requests for important page types, but more importantly: to improve rendering. They presented them explaining that this is specifically to make web pages readable to LLMs, since these can not render content relying on JS. 
Sadly, it's been dismissed as "traditional SEO" and not "shiny new GEO". Until a colleague approached Konny with a new request: To implement "AEO code" - structured data to help LLMs understand and parse what's on the web page.

Yes, Konny is now seeing our community's best SEO therapist.
